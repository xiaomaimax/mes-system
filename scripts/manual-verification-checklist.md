# 系统按钮功能手动验证清单

## 验证目标
验证整个MES系统所有模块的新增、删除按钮功能是否正常工作，确保：
1. 新增按钮能够成功添加数据
2. 添加后数据立即显示在列表中
3. 删除按钮能够正常删除数据
4. 操作有适当的用户反馈

## 验证准备

### 1. 启动系统
```bash
# 启动后端服务
npm run dev

# 启动前端服务
npm run client
```

### 2. 登录系统
- 打开浏览器访问 http://localhost:3000
- 使用管理员账号登录

## 常见问题检查

### 1. 数据不显示问题
如果新增数据后不显示在列表中，检查：
- [ ] 是否使用了useDataService Hook
- [ ] 是否在保存成功后调用了refetch()方法
- [ ] 是否清除了相关缓存
- [ ] 控制台是否有错误信息

### 2. 加载状态问题
- [ ] 是否显示加载动画
- [ ] 加载完成后是否隐藏动画
- [ ] 是否有适当的空状态提示

### 3. 错误处理问题
- [ ] 保存失败时是否显示错误消息
- [ ] 网络错误时是否有重试机制
- [ ] 是否有用户友好的错误提示

### 4. 用户体验问题
- [ ] 操作是否有即时反馈
- [ ] 成功/失败消息是否清晰
- [ ] 表单验证是否正确
- [ ] 按钮状态是否正确（禁用/启用）
