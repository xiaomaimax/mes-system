# 标签页字体大小调整总结

## 调整时间
2024年12月18日

## 调整原因
用户反馈标签页字体太小，影响阅读体验，需要调大字体以提升可读性。

## 调整内容

### 1. ✅ 质量管理模块 (9个标签页)
- **调整前**: font-size: 12px
- **调整后**: font-size: 14px
- **提升**: +2px，提升16.7%

### 2. ✅ 设备管理模块 (7个标签页)
- **调整前**: font-size: 12px
- **调整后**: font-size: 14px
- **提升**: +2px，提升16.7%

### 3. ✅ 生产管理模块 (10个标签页)
- **调整前**: font-size: 11px
- **调整后**: font-size: 13px
- **提升**: +2px，提升18.2%

### 4. ✅ 库存管理模块 (10个标签页)
- **调整前**: font-size: 11px
- **调整后**: font-size: 13px
- **提升**: +2px，提升18.2%

## 调整策略

### 差异化字体大小
- **标签页较少的模块** (质量管理7个、设备管理7个)
  - 使用较大字体: 14px
  - 有更多空间显示

- **标签页较多的模块** (生产管理10个、库存管理10个)
  - 使用适中字体: 13px
  - 平衡可读性和空间利用

### 保持一致性
- 图标字体大小与文字保持一致
- tabBarStyle与内联样式保持同步
- 所有相关样式统一调整
