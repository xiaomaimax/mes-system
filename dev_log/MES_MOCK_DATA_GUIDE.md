# MES系统模拟测试数据使用指南

## 📊 数据概述

为了让MES系统演示更加完整和真实，我们创建了全面的模拟测试数据，涵盖系统的所有核心模块。这些数据不仅数量丰富，而且结构完整，能够支持各种业务场景的演示。

## 🗂️ 数据结构

### 1. 基础数据 (baseData)
- **产品信息**: 5种不同类型的产品（电子产品、汽车零部件、传感器等）
- **生产线信息**: 5条不同类型的生产线（SMT、组装、测试、包装、老化）
- **设备信息**: 8台不同类型的设备，包含状态和利用率
- **物料信息**: 7种物料（PCB、电子元件、芯片、辅料等）
- **员工信息**: 8名员工，涵盖不同部门和岗位

### 2. 生产管理数据 (productionData)
- **生产计划**: 3个生产计划，不同状态（进行中、已完成、计划中）
- **生产任务**: 详细的任务分解，包含操作员、设备、进度等
- **报工数据**: 实际生产数据，包含产量、质量、效率等指标

### 3. 质量管理数据 (qualityData)
- **IQC检验**: 进料检验记录，包含合格率、不良类型
- **PQC检验**: 过程检验数据，实时质量监控
- **FQC检验**: 成品检验记录，最终质量把关
- **不良品记录**: 详细的不良分析和改善措施

### 4. 设备管理数据 (equipmentData)
- **维护记录**: 预防性维护和故障维修记录
- **点检记录**: 日常设备点检数据
- **故障记录**: 设备故障分析和处理记录

### 5. 库存管理数据 (inventoryData)
- **库存信息**: 实时库存状态，包含安全库存预警
- **入库记录**: 采购入库和供应商信息
- **出库记录**: 生产领料和用途追踪

### 6. 人员管理数据 (personnelData)
- **考勤记录**: 员工打卡、加班、迟到等记录
- **培训记录**: 技能培训和认证管理
- **绩效记录**: 员工绩效评估和等级评定

### 7. 工艺管理数据 (processData)
- **工艺路线**: 产品制造工艺流程定义
- **工艺参数**: 关键工艺参数和控制标准

### 8. 系统集成数据 (integrationData)
- **接口配置**: 与ERP、WMS等系统的接口设置
- **同步记录**: 数据同步历史和状态监控

### 9. 报表分析数据 (reportData)
- **KPI指标**: 生产、质量、设备、库存等关键指标
- **趋势数据**: 历史趋势分析数据

## 🛠️ 使用方法

### 1. 导入数据服务
```javascript
import { DataService, DataFormatter, DataCalculator } from '../utils/dataUtils';
```

### 2. 获取数据示例
```javascript
// 获取生产计划
const productionPlans = DataService.getProductionPlans();
const runningPlans = DataService.getProductionPlans('进行中');

// 获取设备信息
const allEquipment = DataService.getEquipment();
const maintenanceRecords = DataService.getMaintenanceRecords('EQ001');

// 获取质量数据
const iqcData = DataService.getIQCInspections('2024-12-22');
const defectRecords = DataService.getDefectRecords();
```

### 3. 数据格式化示例
```javascript
// 格式化日期
const formattedDate = DataFormatter.formatDate('2024-12-22', 'YYYY年MM月DD日');

// 格式化百分比
const efficiency = DataFormatter.formatPercent(95.5, 1); // "95.5%"

// 格式化金额
const amount = DataFormatter.formatCurrency(25500); // "¥25,500"

// 格式化状态
const status = DataFormatter.formatStatus('运行中'); // {color: 'blue', text: '运行中'}
```

### 4. 数据计算示例
```javascript
// 计算生产效率
const efficiency = DataCalculator.calculateEfficiency(950, 1000); // 95%

// 计算合格率
const passRate = DataCalculator.calculatePassRate(98, 100); // 98%

// 计算OEE
const oee = DataCalculator.calculateOEE(96, 92, 98); // 86.7%
```
