### 1. 后端仪表盘数据 API ✅

**文件**: `/opt/mes-system/server/routes/dashboard.js`

**接口列表**:
| 接口 | 功能 | 数据来源 |
|-------|------|---------|
| `GET /api/dashboard/stats` | 统计数据（产量、库存、合格率、异常）| 实时数据库查询 |
| `GET /api/dashboard/production-trend` | 生产趋势（近7天）| production_records |
| `GET /api/dashboard/inventory-distribution` | 库存分布（按物料类型）| inventory |
| `GET /api/dashboard/quality-stats` | 质量统计（合格、返工、报废）| quality_inspections |
| `GET /api/dashboard/equipment-utilization` | 设备利用率 | equipment + production_lines |
| `GET /api/dashboard/export` | 仪表盘数据导出 | Excel 格式 |

**功能特点**:
- ✅ 所有接口需要认证 (`authenticateToken` 中间件）
- ✅ 趋势计算（较上周变化）
- ✅ 支持日期范围查询
- ✅ 参数化 SQL 查询（防止注入）
- ✅ 临时文件自动清理（5秒后删除）
