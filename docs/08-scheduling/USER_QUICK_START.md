# 辅助排程系统 - 用户快速开始指南

**最后更新**: 2025-12-25  
**版本**: 1.0.0

---

## 🚀 5分钟快速开始

### 第1步: 启动系统 (1分钟)

```bash
# 确保你在项目根目录

# 启动后端服务
npm run dev

# 在另一个终端启动前端
cd client && npm start
```

等待两个服务都启动完成：
- 后端: http://localhost:5002
- 前端: http://localhost:3000

### 第2步: 登录系统 (1分钟)

1. 打开浏览器访问 http://localhost:3000
2. 使用默认账号登录
3. 进入系统首页

### 第3步: 导航到排程模块 (1分钟)

1. 在左侧菜单中找到"辅助排程"
2. 点击进入排程模块
3. 你会看到7个标签页

### 第4步: 查看数据 (1分钟)

点击各个标签页查看数据：

| 标签页 | 内容 | 数据量 |
|--------|------|--------|
| 概览 | 统计数据和快速操作 | - |
| 物料 | 6种物料信息 | 6个 |
| 设备 | 5台注塑机 | 5台 |
| 模具 | 7副模具 | 7副 |
| 计划单 | 10个生产计划 | 10个 |
| 任务单 | 排程生成的任务 | 0个(待排程) |
| 排程结果 | 按设备分组的结果 | 0个(待排程) |

### 第5步: 执行排程 (1分钟)

1. 在"概览"标签页中找到"执行排程"按钮
2. 点击按钮
3. 系统会自动为所有未排产的计划单生成任务单
4. 切换到"任务单"标签页查看结果
