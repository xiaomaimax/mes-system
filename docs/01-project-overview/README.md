# MES制造执行系统 - 项目概述

## 🎯 项目简介

MES制造执行系统是一个现代化的制造执行管理平台，旨在为制造企业提供全面的生产过程管理、质量控制、设备管理和数据分析功能。

## ✨ 核心特性

### 🏭 生产管理
- **生产计划管理**: 车间计划制定和资源安排
- **生产执行监控**: 实时生产状态监控和数据采集
- **工单管理**: 生产任务创建、分配和跟踪
- **生产报表**: 生产效率、产量统计和分析

### 🔧 工艺管理
- **工艺路线管理**: 标准化工艺流程定义
- **工艺参数控制**: 关键参数设置和监控
- **工艺文件管理**: 工艺文档版本控制
- **工艺优化**: 持续改进和优化建议

### 🏗️ 设备管理
- **设备档案管理**: 设备基础信息和技术参数
- **设备维护保养**: 预防性维护计划和执行
- **设备故障管理**: 故障记录、分析和处理
- **设备效率分析**: OEE计算和设备利用率统计

### 🔍 质量管理
- **质量检验**: IQC/PQC/FQC/OQC全流程质量控制
- **质量标准管理**: 检验标准和规范维护
- **不良品管理**: 缺陷记录、分析和改进
- **质量追溯**: 产品质量全程可追溯

### 📦 库存管理
- **物料管理**: 原材料、半成品、成品库存管理
- **出入库管理**: 物料出入库操作和记录
- **库存盘点**: 定期盘点和差异处理
- **库存分析**: 库存周转率和成本分析

### 👥 人员管理
- **员工信息管理**: 员工档案和基础信息维护
- **考勤管理**: 员工考勤记录和统计
- **培训管理**: 员工技能培训和认证
- **绩效管理**: 员工绩效评估和激励

### 🔗 系统集成
- **数据接口管理**: 与ERP、WMS等系统数据交换
- **设备数据采集**: 生产设备数据实时采集
- **第三方系统对接**: 支持多种标准协议
- **消息推送**: 企业微信、邮件等多渠道通知

### 📊 报表分析
- **综合看板**: 生产、质量、设备关键指标展示
- **专业报表**: 各模块专业数据分析报表
- **自定义报表**: 用户自定义报表设计和生成
- **KPI分析**: 关键绩效指标监控和分析

### ⚙️ 系统设置
- **用户权限管理**: 用户账号、角色和权限控制
- **部门权限配置**: 不同部门访问不同功能模块
- **系统参数配置**: 系统运行参数和业务规则设置
- **安全管理**: 系统安全策略和审计日志

## 🎨 技术特点

### 前端技术
- **React 18**: 现代化前端框架
- **Ant Design**: 企业级UI组件库
- **Recharts**: 专业数据可视化
- **React Router**: 单页应用路由管理

### 后端技术
- **Node.js**: 高性能服务端运行环境
- **Express**: 轻量级Web应用框架
- **MySQL**: 可靠的关系型数据库
- **RESTful API**: 标准化接口设计

### 架构特点
- **模块化设计**: 松耦合的模块化架构
- **响应式布局**: 适配PC、平板、手机多端
- **实时数据**: WebSocket实时数据推送
- **微服务架构**: 支持分布式部署
