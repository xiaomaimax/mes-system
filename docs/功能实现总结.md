### 1. 数据导入/导出功能

#### 后端 API
**文件**: `/opt/mes-system/server/routes/import-export.js`

**功能**:
- ✅ **批量导入** (POST `/api/import-export/:module/import`)
  - 支持 Excel (.xlsx, .xls) 和 CSV 文件
  - 文件大小限制：10MB
  - 自动字段过滤（只插入表中存在的列）
  - 批量插入 + 重复键更新
  - 返回导入结果（成功数、失败数）

- ✅ **数据导出** (GET `/api/import-export/:module/export`)
  - 支持 Excel 和 CSV 格式
  - 最多导出 10,000 条记录
  - 临时文件自动清理（5秒后删除）
  - 进度条实时反馈

- ✅ **模板下载** (GET `/api/import-export/:module/export/template`)
  - 自动生成导入模板（包含所有字段注释）
  - 支持所有模块（materials, suppliers, warehouses 等）

**依赖包**:
- xlsx ^0.18.5 - Excel 读写
- multer ^1.4.5 - 文件上传

## 🔧 技术实现细节

### 导入功能工作流
```
1. 用户点击"导入"按钮
2. 打开模态框，用户选择文件
3. 前端验证文件格式和大小
4. 上传到 `/api/import-export/:module/import`
5. 后端使用 xlsx 解析文件
6. 过滤字段（只保留表中存在的列）
7. 批量插入数据库（ON DUPLICATE KEY UPDATE）
8. 返回导入结果（成功数、失败数）
9. 前端显示结果和刷新列表
```

### 导出功能工作流
```
1. 用户点击"导出"按钮
2. 打开模态框，用户选择格式（Excel/CSV）
3. 请求 `/api/import-export/:module/export?format=xxx`
4. 后端查询数据（最多 10,000 条）
5. 使用 xlsx 生成文件
6. 保存到临时目录
7. 发送文件给浏览器下载
8. 5 秒后删除临时文件
```

### 图表渲染流程
```
1. 页面加载时调用 `fetchDashboardData()`
2. 并发请求多个数据接口
3. 显示加载状态
4. 数据返回后渲染图表
5. Recharts 响应式容器自动适配
6. 用户可以刷新或改变日期范围
```
