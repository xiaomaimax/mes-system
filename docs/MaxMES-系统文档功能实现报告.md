## 📝 内容管理

### 更新日志维护

**位置**: `SystemDocs.js` 中的 `updateLogs` 数组

**格式**:
```javascript
{
  version: 'v1.5.0',
  date: '2026-04-04',
  type: 'feature',
  title: 'UI/UX 优化项目完成',
  changes: [
    '完成 Phase 1-5 全面优化',
    '新增 PWA 支持',
    // ...
  ]
}
```

**添加新版本**:
1. 在 `updateLogs` 数组开头添加新对象
2. 选择类型：feature/performance/fix/release
3. 填写变更列表

### 文档列表维护

**位置**: `SystemDocs.js` 中的 `docCategories` 数组

**添加新文档**:
```javascript
{
  title: '新分类',
  icon: '📗',
  docs: [
    { name: '文档名称', path: '/docs/xxx.md', tag: '标签' }
  ]
}
```
